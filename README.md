# FastNotes - IOS
Fast note is a simple to use notes app.

## Introduction
This notes app is a simple and easy to use app created with the main focus on user friendly. It's packed with the most basic functions of a notes app to make sure the app stays uncluttered. It's mainly ment to take down notes on the go or in meetings if you just need a rough note to further elaberate on later.


## Key features
- Create note
- Edit note
- Remove note

## Language
This app is written in react with a backend on supabase

## How to use
Start off by first starting the software. This is done by launching expo go through terminal with the command *npm expo start*.

Then you scan the QR code with your phone (**Expo go must be installed on your device**) and it will launch the app.

Once the app is open, to create a new note you simply press the *new note* button and fill in both title and note-message before proceeding to press *create*

To inspect your note you can press the title showcased in your notes list.
