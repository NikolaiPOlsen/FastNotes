# FastNotes - IOS
Fast note is a simple to use notes app.

assignment_4
## Introduction
This notes app is a simple and easy to use notes app created with the main focus on user friendlyness. It's packed with the most basic functions of a notes app to make sure the app stays uncluttered. It's mainly ment to take down notes on the go or in meetings if you just need a rough note to further elaberate on later.

## Key features
- Create note
- Edit note
- Remove note

## Language
This app is written in react native with a backend on supabase

## How to use
1. Start the app with:
   ```bash
   npm expo start
   ```

2. Open **Expo Go** on your device.

3. Scan the QR-Code shown in the terminal

---

## Create a note
1. Click **New Note**  

2. Fill inn the following:
   - Title
   - Note 
   - Add a photo (not mandatory)

3. Click **Create**
